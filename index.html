What I've Learned In Coding Class

Hello, it's Emma here, and this blog post is about what I've learned about coding this year. Before I started coding class this year, the only impression I had of coding was the green numbers running down the screen in The Matrix: Now I know a thousand times more about coding than I did then. Ok, maybe not quite a thousand, but at least several hundred. Today I'll tell you about some of the things I've learned.

These are the three coding languages I've learned so far:

  • JavaScript
  • HTML
  • CSS

In JS I've learned how to:

  • Create shapes
  • Add color to the shapes
  • Create variables
  • Animate
  • How computer mice can interact with a drawing program

In HTML I've learned how to:

  • Create tags
  • Edit text using tags
  • Create lists
  • Insert internal and external links into text
  • Create tables

In CSS I've learned how to:

  • Use CSS selectors
  • Use CSS properties

To Elaborate:

HTML and CSS go hand-in-hand with each other. CSS basically modifies HTML; it adds fancy stuff like fonts, color, text size, etc. HTML and CSS are the programming languages used to create websites. JavaScript is used for creating computer programs like games and other apps.

Tags:

Tags are what create paragraphs and headings and links and everything else in HTML.

Lists:

There are two types of lists: ordered and unordered. Ordered lists are numbered lists, while unordered lists are bullet points.

Tables:

Tables are really complicated to make, and they require a lot of tag memorization.

CSS Selectors:

They select all of an element (e.g. all paragraphs) or and id or class in order to modify it.

CSS Properties:

Add things like color and font.

Ids and Classes:

You can add an id to a certain paragraph or heading or whatever, and then use CSS to modify only the tag with that specific id. A class is pretty much an id added to multiple tags; it allows you to modify several things in the same way.

Links:

There are internal and external links. Internal links take you to a certain place within the page that you're in, while external links take you to different page entirely.

So basically, I've learned a lot, but I obviously haven't learned nearly everything. There is always more to know, and I will continue learning.
